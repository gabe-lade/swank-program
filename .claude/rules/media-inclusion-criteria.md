# Media Inclusion Criteria (`_data/media.yml`)

> **Status: DRAFT for Gabe's review.** Edit tiers and outlet lists freely;
> agents apply whatever this file says.

An item must clear **both** gates: Relevance AND Outlet tier.

## Gate 1 — Relevance (at least one)

- **R1.** Quotes or interviews Gabriel Lade (any affiliation wording: OSU,
  AEDE, Ohio State, Swank Chair; formerly Iowa State).
- **R2.** Names Gabriel Lade or the Swank Program / Swank Chair in the
  article body. A byline-only or footnote mention does not count.
- **R3.** Covers, by name, a paper in `_data/publications.yml` or a Swank
  Program event/report (e.g., the biofuels forum).

## Gate 2 — Outlet tier

**Tier 1 — include.**
- Wire/national: AP, Reuters, NYT, WaPo, NPR, Bloomberg, WSJ.
- Ohio news: Columbus Dispatch, Cleveland.com, Spectrum News 1, Ohio Capital
  Journal, Statehouse News Bureau, Toledo Blade, Dayton Daily News.
- Public media / ag radio: Harvest Public Media, MPR, Brownfield Ag News.
- Established trade press: Ohio's Country Journal / Ohio Ag Net, Agri-Pulse,
  DTN Progressive Farmer, Farm Progress, Successful Farming, Ethanol
  Producer Magazine, Ohio Farm Bureau (ofbf.org).
- Podcasts with named hosts and editorial production.

**Tier 2 — include, but note it in the PR body.**
- Small-market local papers (e.g., Northwest Signal).
- Editorially independent campus media (The Lantern).
- Topical aggregators that add their own framing (Advanced Biofuels USA).
- Syndicated reprints when the originating outlet is NOT already listed.

**Tier 3 — exclude.**
- OSU/AEDE/CFAES press releases and verbatim reposts of them.
- SEO content farms / AI-spun rewrites (signs: no byline, scraped layout,
  keyword-stuffed domain).
- Pure aggregator duplicates of an item already included.
- Social media posts, event calendars/listings, dead links.

**Tier UNKNOWN — do not decide.** Unfamiliar outlet → mark `Tier-UNKNOWN`
and escalate to the human.

## Dedup / syndication

- Same story syndicated across outlets → keep the **originating** outlet
  only; list the copies in the PR notes. (Precedent: the Biofuels Forum
  story appears under both Advanced Biofuels USA and Northwest Signal in
  `media.yml` — that was a deliberate human call. When in doubt, ask.)
- Same `url` or same `(title, source)` as an existing entry → skip.
